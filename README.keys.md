# 按鍵說明

## 來自原本 MS Edit 的按鍵

| 按鍵     | 說明               |
| -------- | ------------------ |
| Ctrl + N | 建立新檔案         |
| Ctrl + O | 開啟舊檔           |
| Ctrl + S | 存檔               |
| Ctrl + W | 關閉目前檔案       |
| Ctrl + Q | 離開               |
| Ctrl + Z | 還原               |
| Ctrl + Y | 重作               |
| Ctrl + X | 剪下               |
| Ctrl + C | 複製               |
| Ctrl + P | 貼上               |
| Ctrl + F | 尋找               |
| Ctrl + R | 取代               |
| Ctrl + A | 全選               |
| Ctrl + P | 前往特定檔案       |
| Ctrl + G | 前往特定行/列      |
| Ctrl + L | 選擇目前所在行     |
| Alt + 上 | 將目前所在行往上移 |
| Alt + 下 | 將目前所在行往下移 |

## 新增按鍵

| 按鍵      | 說明               |
| --------- | ------------------ |
| F1 或 F10 | 叫出選單           |
| F2        | 存檔               |
| F3        | 存檔後離開         |
| F4        | 離開               |
| Ctrl + K  | 在目前行刪除到行尾 |
| ESC | 切換到命令列 |

## 中文標點符號

| 按鍵                  | 說明          |
| --------------------- | ------------- |
| alt + ,               | 輸入逗號 ，   |
| alt + , 兩下          | 輸入括弧 〈   |
| alt + .               | 輸入句號 。   |
| alt + . 兩下          | 輸入括弧 〉   |
| alt + ;               | 輸入分號 ；   |
| shift + alt + ;       | 輸入冒號 ：   |
| alt + '               | 輸入頓號 、   |
| alt + \[              | 輸入引號 「   |
| alt + \]              | 輸入引號 」   |
| alt + \[ 兩下         | 輸入引號 【   |
| alt + \] 兩下         | 輸入引號 】   |
| shift + alt + \[      | 輸入 『       |
| shift + alt + \]      | 輸入 』       |
| shift + alt + \[ 兩下 | 輸入 〖       |
| shift + alt + \] 兩下 | 輸入 〗       |
| alt + 1               | 輸入驚嘆號 ！ |

## 命令列

| 命令 | 說明 |
| -- | -- |
| 數字 | 跳到指定行 |
| s/<關鍵字> | 搜尋 |
| s/<關鍵字>/<關鍵字> | 取代 |
